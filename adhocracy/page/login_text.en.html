In order to register, click on "Login" or "Register" in the upper right corner.


Now fill in all fields under "Register". You can choose your user name and change it later if you want; keep in mind that it will be displayed publicly. Click on the Register button.


You are now registered at liqd.net. Welcome!
If you want to change your profile settings, click on "Settings" in the upper right corner.


On this profile settings page you can set your preferences:

  1. (a) the language in which you want liqd.net to be displayed for you
  2. (b) You can set your individual avatar picture using the "Gravatar" service.
  3. You can change your password (c).
  4. You can change the e-mail address under which you want liqd.net to get in touch with you (d).


In the text field (a) you can write something about yourself, point others to interesting subjects and link to other websites such as blogs, Facebook profiles etc. Click (b) for some formatting help - it will open in a separate window (c)
Don't forget to save your new settings by clicking on (d).
You can change and add to your settings at any time.


Now you can join the instances that you are interested in.
You can view a list of all instances on liqd.net if you click on Instances in the upper left corner.


This gets you to an overview page. You can sort the list by different criteria.
At the bottom of the list you can set the number of instances displayed on one page, and you can navigate to the next page if there are more instances than can be displayed on one page.
Just click on an instance's name to get more information about it.


Now you are on the home page of an instance you are interested in. To join it you just have to click the green join button on the right; now you can get involved with discussions and polls in your instance! You can easily join more instances if you want.
